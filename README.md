# AI Annotation & Evaluation Portfolio

This repository demonstrates my ability to evaluate, annotate, and improve AI-generated responses using structured rubrics aligned with industry standards.

## Key Capabilities
- AI response evaluation (accuracy, clarity, helpfulness, safety)
- Data annotation and labeling
- Prompt engineering and edge-case testing
- Hallucination detection and correction

## Included Work
- Annotated datasets
- Evaluation rubrics
- Real examples of AI output improvements
- Edge case and failure analysis

## Methodology
Each AI response is:
1. Evaluated using a structured rubric
2. Issues identified (factual, logical, safety)
3. Improved with a corrected version

## Tools Used
ChatGPT, Claude, Google Docs, Excel

## Goal
To contribute high-quality data that improves AI model performance and reliability.
